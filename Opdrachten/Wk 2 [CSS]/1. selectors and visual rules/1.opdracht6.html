Child selectors

div > p { color: red; }

The child selector selects all elements that are the children of a specified element. In my example it allows one to apply styles to paragraphs that are children of divs.

Adjacent selectors

h2 + p { color: green; }

The adjacent sibling selector is used to select an element that is directly after another specific element.
Sibling elements must have the same parent element, and "adjacent" means "immediately following". Similar to child selectors, though it allows one to apply styles to adjacent sibling elements.

General sibling selector

h2 ~ p { color: blue; }

The general sibling selector selects all elements that are siblings of a specified element. Similar to the adjecent sibling selector, but works on all siblings

Conclusion: when used correctly combinators can be used to reduce the ammount of CSS needed to style a page.
